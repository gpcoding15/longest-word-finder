# longest-word-finder

A small JavaScript utility that returns **the longest word** from a given string.  
Includes automated tests written with **Jest**.

---

## Project Structure

- `longest-word-finder.js` – Contains the main function that finds the longest word.
- `longest-word-finder.test.js` – Contains Jest test cases for the function.

> This README assumes the exported function is named `findLongestWord`.  
> If your function has a different name, simply update the examples.

---

## Requirements

- Node.js (v14 or higher recommended)
- npm or yarn

---

## Installation

Clone the repository and install dependencies:

```bash
npm install
```

### If Jest is not installed yet, add it with:

 npm install --save-dev jest

### Run all Jest tests with:

 npm test
